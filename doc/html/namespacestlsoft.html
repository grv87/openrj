stlsoft Namespace Reference

Detailed Description

The Open-RJ project inserts c_str_ptr, c_str_data and c_str_len string access shims into the STLSoft namespace, for generalised manipulation of its ORJStringA, ORJRC and ORJ_PARSE_ERROR types.
