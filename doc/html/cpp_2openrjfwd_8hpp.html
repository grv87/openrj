openrj/cpp/openrjfwd.hpp File Reference

Detailed Description

[C++ only] Forward declarations for the Open-RJ/C++ mapping.

This file is useful for the case where you wish to implement your components in terms of types of the openrj::cpp namespace, but do not need to include the full definitions in order to define the components.
